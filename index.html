HOME MANAGER

Home Manager

React web application that we made durging the Junior Front-end Developer bootcamp organized by infoShare Academy in a group of three girls. Each of us was reponible for a different part of the project. We worked in SCRUM using JIRA software

The purpose of the developed application is to manage tasks added by users. MVP version was designed for Mobile Devices.

Available functionality:

  • Authentication with Firebase
  • Add task form enables adding tasks to Firebase collection for a logged user
  • Display tasks added to Firestore
  • Expand chosen task to read more information
  • Update task data thanks to edit functionality
  • Delete task and removal from Firebase collection
  • Search of tasks by phrase
  • Filter tasks by category
  • Clearing all set filters on-button-click
  • Display a calendar dowloaded from React Big Calendar
  • Setup of user name that will be displayed on a Home page
